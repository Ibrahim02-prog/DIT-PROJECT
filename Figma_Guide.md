# Figma Design Guide

## Purpose

This document explains how to navigate and use the AgriSys Figma prototype.

---

## Navigation

The prototype uses interactive links.

Buttons automatically navigate between screens.

Back buttons return users to the previous screen.

Cards open detailed pages.

Menu items navigate to their respective modules.
